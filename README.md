# דשבורד עלויות · מעבדת צנתורים (Cath Lab Cost Dashboard)

דשבורד אינטראקטיבי לניתוח עלויות פרוצדורות במעבדת הצנתורים — בית חולים שיבא.

הדשבורד הוא קובץ HTML עצמאי יחיד (`index.html`) — כל הנתונים מוטמעים בתוכו, ללא תלות באינטרנט או בשרת. ניתן לפתוח אותו ישירות בדפדפן.

## מה כולל

- **כרטיסי KPI**: סה״כ פרוצדורות, עלות כוללת, עלות ממוצעת לפרוצדורה, מספר פריטים, רופאים וסוגי פרוצדורות.
- **פרוצדורות לפי רופא** — גרף עם מתג בין מספר פרוצדורות / עלות כוללת / עלות ממוצעת.
- **פרוצדורות לפי סוג** — 15 הסוגים המובילים.
- **טבלת פירוט פרוצדורות** עם מיון, חיפוש ודפדוף.
- **צלילה לעומק (drill-down)** — לחיצה על פרוצדורה פותחת את כל הפריטים עם המחירים (תיאור, מק״ט בי״ח, מק״ט יצרן, כמות, מחיר ליחידה, עלות שורה, אופן הצלבת מחיר) + סה״כ עלות הפרוצדורה.
- **מתג כולל / ללא מע״מ** המשפיע על כל המספרים.
- **ייצוא CSV** של התצוגה הנוכחית.

## נתונים

הופק מתוך קובץ מקור פנימי (Excel). קבצי המקור **אינם** נכללים ב-repo זה (ראו `.gitignore`).

> ⚠️ **חיסיון**: הדשבורד מכיל שמות רופאים ועלויות פנימיות. ה-repo מוגדר כ-**פרטי**. אין לפרסם בפומבי ללא הסרת/מיסוך מידע מזהה.

## הרצה מקומית

פתחו את `index.html` בדפדפן, או הריצו שרת סטטי:

```bash
python -m http.server 8090
# גשו אל http://localhost:8090
```
